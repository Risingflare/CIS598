Welcome to my File Transfer Tool and Store Creator

This program was designed with Cheers POS in mind

At the current state that this is written the following program can:

  1. Take a csv file, parse, and get item data
  2. Create a store with that item data and enter its items
  3. Create an ever growing Item Library
  4. The Item library is used to fill in some of the blank columns of the csv if possible
  5. You can create a new store and select items from the Item Library to add to your store
  6. Distributor list will grow with every new Distributor found
  7. Size list will grow with every new Size found
  8. Category list will grow with every new Category found
  9. Lastly, you can download a store's data as a csv if you want to modify its data in your own manner
